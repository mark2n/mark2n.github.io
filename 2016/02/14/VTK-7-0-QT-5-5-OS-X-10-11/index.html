Installing QT

Compiling and installing VTK

  • Download VTK source package .zip(from http://fossies.org/linux/misc/VTK-7.0.0.tar.gz/), and unpack it(or git it)

  • Install camke(I had CMake GUI and cmake installed from homebrew. I don’t know which tool I am really using. I guess the one from homebrew will do the work.)
